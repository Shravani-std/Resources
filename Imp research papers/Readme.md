## 📚 Research Papers & References

The following research papers provide theoretical and practical foundations for the concepts explored in this project, including AI agent memory, multimodal agents, vector embeddings, efficient language models, and graph-based memory systems.

### 🧠 Understanding How LLMs Work

* **[How LLMs Actually Work](https://www.0xkato.xyz/how-llms-actually-work)**
  A practical walkthrough of the internal architecture of modern Large Language Models (LLMs), covering tokenization, embeddings, positional encoding, self-attention, multi-head attention, feed-forward networks, residual connections, normalization, and next-token prediction.

### 🧠 AI Agent Memory

* **[Memory in the Age of AI Agents](https://arxiv.org/pdf/2512.13564)**
  A comprehensive survey of memory systems for AI agents, covering memory forms, functions, lifecycle dynamics, benchmarks, and future research directions.

### 🤖 Multimodal AI Agents

* **[Magma: A Foundation Model for Multimodal AI Agents](https://arxiv.org/pdf/2502.13130)**
  Introduces a multimodal foundation model designed for AI agents capable of understanding images, videos, user interfaces, and robotic environments while supporting action planning and execution.

### 🔢 Vector Embeddings and Semantic Representation

* **[Efficient Estimation of Word Representations in Vector Space](https://arxiv.org/abs/1301.3781)**
  A foundational paper on efficient word-vector learning that introduces architectures for representing words in continuous vector spaces while preserving semantic and syntactic relationships.

### ⚡ Efficient Models for Agentic AI

* **[Small Language Models are the Future of Agentic AI](https://arxiv.org/abs/2506.02153)**
  Discusses why Small Language Models (SLMs) can be more efficient and economical than large general-purpose models for repetitive and specialized tasks within agentic AI systems.

### 🕸️ Graph-Based Memory for LLM Agents

* **[Memory is Reconstructed, Not Retrieved: Graph Memory for LLM Agents](https://arxiv.org/abs/2606.06036)**
  Presents a graph-based memory framework where agents dynamically reconstruct relevant memories through associative connections instead of relying only on traditional static retrieval pipelines.

---

> These papers serve as key references for understanding the evolution of intelligent AI systems—from semantic vector representations to multimodal agents, efficient agentic models, and advanced long-term memory architectures.
