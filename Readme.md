### System Design Primer

**Source:** [https://github.com/donnemartin/system-design-primer](https://github.com/donnemartin/system-design-primer)
One of the most popular system design repositories on GitHub (~330k+ stars). It teaches how to design **large-scale distributed systems** and prepare for system design interviews with examples, patterns, and practice questions. ([GitHub][1])

---

### System Design 101

**Source:** [https://github.com/ByteByteGoHq/system-design-101](https://github.com/ByteByteGoHq/system-design-101)
A beginner-friendly resource that explains **system design concepts using diagrams and visual explanations**. Best for understanding fundamentals like APIs, caching, networking, and distributed systems.

### System Design (Karan Pratap Singh)

**Source:** [https://github.com/karanpratapsingh/system-design](https://github.com/karanpratapsingh/system-design)
An open-source **system design course with structured notes**. Covers architecture basics, scalability concepts, and practical system design topics in a simple learning path.

### Awesome System Design Resources

### System Design Interview

**Source:** [https://github.com/checkcheckzz/system-design-interview](https://github.com/checkcheckzz/system-design-interview)
Focuses on **common system design interview problems** and their approaches. Helps understand how to design systems like messaging services, social networks, and scalable platforms.

### System Design Academy

### InterviewReady System Design Resources
